# Playwright Evidence

Prefer evidence that another engineer can reproduce.

Useful evidence:

- Screenshot path or description.
- Console errors and warnings.
- Failed network requests, status codes, and payload hints.
- Accessibility snapshot for important controls and labels.
- DOM state for disabled, selected, validation, loading, and error states.
- Storage/cookie/session observations for persistence or auth issues.
- Test, build, and dev-server output when relevant.

When screenshots or traces are not saved, describe exact route, viewport,
interaction sequence, expected behavior, and observed behavior.
