# Large Output

When output may flood context:

- Search first, then read targeted ranges.
- Summarize command output by error class, file, and actionable detail.
- Save or reference large artifacts instead of pasting them into conversation.
- Use structured parsers for JSON, CSV, logs, and test output when available.
- Keep raw evidence accessible for later verification.

Do not let compression erase the exact error lines needed to reproduce a
finding.
